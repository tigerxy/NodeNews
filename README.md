# NodeNews
Not more than a cool and selfhosted Newspage. Using the [Cognitive Services](https://www.microsoft.com/cognitive) Bing API.

## Installation
Clone the repository or download the files.

```git clone https://github.com/tigerxy/NodeNews.git ```

```cd ./NodeNews```

Install the node.js dependencies.

```npm install```

Insert your Subscription Key in the main.js. You get it [here](https://www.microsoft.com/cognitive-services/en-US/subscriptions).
## Run
Start the application.

```node main.js```

Open your Browser and navigate to:

``` http://localhost:8080/ ```
